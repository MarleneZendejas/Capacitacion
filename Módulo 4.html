Páginas WEB Diseño Digital

En el submódulo Páginas WEB creamos una página web con el lenguaje HTML empleando títulos, textos, tablas, listas, imágenes, entre otras, implementandolo con otro archivo CSS para que éste se vea más estético y con mejor presentación, combiando el tamaño de la letra, el color, estilo, alineación, tablas con diseño, entre otras cosas.





En el submódulo Diseño Digital utilizamos el programa CorelDraw, donde aprendimos algunas herramientas que nos ayudan a realizar algun diseño con mucha más facilidad así como crear figuras y poder modificar esta para así adaptar lo que se quiere conseguir. También hay herramientas para pintar así como crear un degradado o un color sólido. Todas estas herramientas te ayudan para llevar un mejor manejo de nuetro diseño y éste tenga una mejor presentación
